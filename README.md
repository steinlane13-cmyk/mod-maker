# Crimson Heart Blade Mod

A Minecraft mod that adds the Crimson Heart Blade sword to the game.

## Features

- **Crimson Heart Blade Sword**
  - 15 attack damage
  - 900 durability
  - Grants invisibility to player and armor while held
  - Glowing crimson appearance

## Installation

1. Place the mod files in your Minecraft mods directory
2. Ensure Forge is installed
3. Launch Minecraft

## Recipe

Craft the Crimson Heart Blade using:
- 3 Crimson Ingots (vertical column)
- 1 Stick (center of bottom row)

## Usage

Hold the sword in your hand to become invisible. Release to become visible again.
