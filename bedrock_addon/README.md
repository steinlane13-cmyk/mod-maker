# Crimson Heart Blade - Bedrock Addon

A custom Minecraft Bedrock addon that adds the legendary Crimson Heart Blade sword.

## Installation

### Step 1: Download
Clone or download this repository

### Step 2: Import Addon
1. Copy the `bedrock_addon` folder to your Bedrock addons directory:
   - **Windows 10/11**: `%appdata%\.minecraft\com.microsoft.minecraftuwp\minecraftWorlds\[Your World]\behavior_packs` and `resource_packs`
   - **Mobile**: Check your app's games folder for `bedrock` directory
   - **Console**: Use the in-game marketplace or addon manager

### Step 3: Enable in World
1. Open/create a new Minecraft world
2. Go to World Settings
3. Activate both behavior and resource packs
4. Enable experimental features if needed

## Sword Specifications

- **Name**: Crimson Heart Blade
- **Attack Damage**: 15
- **Durability**: 900 hits
- **Special Effect**: Grants invisibility to player and armor while held
- **Repair Material**: Crimson Blocks
- **Enchantable**: Yes (up to 10 levels)

## Crafting Recipe

```
C
C
S
```

Where:
- **C** = Crimson Block (2 required)
- **S** = Stick (1 required)

Craft at a Crafting Table
