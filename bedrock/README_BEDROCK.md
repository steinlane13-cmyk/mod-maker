# Crimson Heart Blade - Bedrock Edition

This is the Bedrock Edition version of the Crimson Heart Blade mod.

## Installation for Bedrock

### Windows 10/11
1. Download the mod files
2. Navigate to: `%appdata%\.minecraft\com.microsoft.minecraftuwp\minecraftWorlds`
3. Create a new world or open an existing one
4. Go to `behavior_packs` and `resource_packs` folders
5. Copy the bedrock folder contents to both directories
6. Open Minecraft and activate the pack in world settings

### Mobile (iOS/Android)
1. Download the mod files
2. Place them in your app's `games/bedrock` directory
3. Create a new world and enable the pack

### Console (Xbox, Switch, PlayStation)
1. Use a marketplace or mod manager compatible with your platform
2. Install the behavior and resource packs
3. Enable in world settings

## Features
- ⚔️ 15 attack damage
- 🛡️ 900 durability
- 👻 Invisibility while held
- 🔧 Repairable with Crimson Blocks
- 📱 Works on all Bedrock platforms

## Crafting Recipe

```
C
C
S
```

Where **C** = Crimson Block, **S** = Stick

## Compatibility
- Minimum Bedrock version: 1.20.0+
- Works on: PC, Mobile, Console, Switch, VR
